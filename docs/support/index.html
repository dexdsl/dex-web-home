FAQ

Why does a page sometimes look stale after a soft navigation?

If a route runtime fails mid-transition, refresh state can desync. Use direct route reload and then continue navigation from the rendered route.

Can I use support without signing in?

Yes. Support is fully public. Signing in only unlocks account-specific shortcuts.

How often does the status panel refresh?

Status refreshes every 60 seconds when healthy and uses exponential backoff during outages or endpoint failures.
